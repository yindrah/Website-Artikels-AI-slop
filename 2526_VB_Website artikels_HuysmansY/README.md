# Reactiviteit: UX/UI Ontwerp en gedrag

**Student:** Huysmans Yindra  
**Klas:** 1GDM2

## Project
Deze Next.js-website bevat drie afzonderlijke artikelpagina's op basis van artikels van De Standaard. Per artikel is er een samenvatting, inhoud en UX/UI-analyse met vijf sterke punten en vijf verbeterpunten.

## UX/UI-keuzes
- Grote hero-afbeeldingen zorgen voor een duidelijke eerste indruk.
- Een vaste header maakt navigeren eenvoudig.
- Korte alinea's en ruime witruimte verbeteren de leesbaarheid.
- Analysekaarten maken sterke punten en verbeterpunten scanbaar.
- Tailwind CSS wordt gebruikt voor consistente spacing, typografie en responsive layout.

## Starten
```bash
npm install
npm run dev
```

Open daarna `http://localhost:3000`.
